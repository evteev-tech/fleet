## Шаг 3 — Применить в проект

1. Скопировать все 8 файлов из `analytics_modules/` (кроме `analytics.js`) в `js/screens/analytics/`
2. Заменить `js/screens/analytics.js` на новый главный файл

**Структура после рефакторинга:**
```
js/screens/
  analytics.js          ← главный оркестратор
  analytics/
    utils.js           ← общие утилиты
    overview.js        ← вкладка "Обзор"
    opex.js            ← вкладка "Расходы"
    capex.js           ← вкладка "CAPEX"
    pnl.js             ← вкладка "По машинам"
    kassas.js          ← вкладка "Кассы"
    forecast.js        ← вкладка "Прогноз"
    desktop.js         ← Desktop layout
```
